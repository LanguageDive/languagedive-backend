# LanguageDive

> **Learn languages by reading what you love.**
>
> Full documentation in Spanish → [`README.es.md`](./README.es.md)

LanguageDive is an immersive reading platform that turns real books (EPUB) into a language learning experience. Import any book, read sentence by sentence, and build your personal vocabulary — naturally, through context.

### Quick start

```bash
docker compose up
```

API at `http://localhost:8080`, Swagger UI at `/swagger-ui/index.html`.

### Tech stack

| Layer | Technology |
|---|---|
| Language | Java 25 |
| Framework | Spring Boot 4.0 |
| Database | PostgreSQL 17 |
| Migrations | Flyway |
| Auth | Spring Security + JWT |
| Build | Maven Wrapper |
