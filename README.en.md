# LanguageDive

**Learn languages by reading what you love.**

LanguageDive is an immersive reading platform that turns real EPUB books into a language learning experience. Upload any book, read sentence by sentence, and build your personal vocabulary naturally through context — without losing track of your progress.

Stop studying. Start reading.

## Features

- Import any EPUB file — auto-detects language, splits into sentences and chapters
- Read sentence by sentence with adjustable page sizes
- Tap any word to save it to your personal grimoire with translation
- Track your vocabulary learning status: new → learning → mastered → ignored
- Automatic progress saving per sentence — never lose your place

## Quick start

```bash
docker compose up
```

API at `http://localhost:8080`, interactive docs at `/swagger-ui/index.html`.

## Tech stack

Java 25, Spring Boot 4.0, PostgreSQL 17, Flyway, Docker.
